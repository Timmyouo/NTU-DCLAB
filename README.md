# dclab-labs

Supporting labs from a **digital circuits / FPGA** course (SystemVerilog, **Altera DE2-115**). Each directory is one assignment; see the README inside for scope and tooling.

| Lab | Directory | Topics |
|-----|------------|--------|
| 1 | [`lab01_fpga_intro/`](lab01_fpga_intro/) | Synthesizable Verilog style, simple FSM, seven-segment / DE2-115 bring-up |
| 2 | [`lab02_rsa256/`](lab02_rsa256/) | RSA-256 modular exponentiation, Avalon-MM master, RS-232 to host |
| 3 | [`lab03_audio_i2c/`](lab03_audio_i2c/) | I²C init for WM8731, audio record/playback with rate change |

These are **student implementations** for coursework. Long-form lab handouts from the instructor are not duplicated here; only short project descriptions are included.
