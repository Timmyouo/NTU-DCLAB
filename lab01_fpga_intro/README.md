# Lab 1 — FPGA intro (DE2-115)

**Goal:** Practice **synthesizable SystemVerilog** (registered vs combinational logic, naming), simple **FSM**, and basic **DE2-115** I/O (keys, seven-segment displays).

**Contents:** `src/Top.sv` and DE2-115 wrapper under `src/DE2_115/`; simulation helpers under `sim/`.

**Tools:** Quartus for synthesis; VCS (or your course simulator) where `tb_*.sv` is used.

Course-wide coding guidelines were provided separately; this folder keeps the implementation only.
